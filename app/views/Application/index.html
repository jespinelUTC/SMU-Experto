Evaluación heurística ¿Para que sirve?

Un Análisis Heurístico, o Evaluación Heurística es una técnica que se emplea para buscar errores de usabilidad principalmente en sitios web, aunque también puede utilizarse para evaluar cualquier tipo de aplicación o sistema que tenga una interacción con usuarios.
La evaluación consiste en la verificación del cumplimiento de un conjunto de parámetros que aseguran la usabilidad del sistema que se este analizando, para luego establecer un resultado de la evaluación.
SMU-Experto es una herramienta que facilita esta tarea a los profesionales ligados a la calidad del software ya que registra las correspondientes puntuaciones dadas a cada uno de los parámetros y de manera automática genera el resultado del test.
